Unity 3D Platformer – Collectibles Game

This project is a simple 3D platformer built in Unity as part of my learning and experimentation with game development.
The gameplay involves controlling a robot character, exploring platforms, and collecting items while navigating through a custom-designed environment.

Features

3D character movement and controller

Collectible system with real-time UI counter

Multiple platforms, ramps, and obstacles

Custom environment with lighting and particle effects

HUD showing remaining collectibles

WebGL build support

Beginner-friendly code structure

Technologies Used

Unity (2022/2023)

C# scripting

WebGL Build

Basic VFX, materials, and lighting

How to Play

Move the player using keyboard input (WASD / arrow keys).

Explore the level and collect all the collectibles.

The UI shows the remaining collectibles.

Complete the level when the counter reaches zero.
